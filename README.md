## 🧠 AI Decision Engine

> The brain of SevaSetu — turning raw data into actionable strategies.

### How it Works

1. NGO uploads **CSV/PDF reports**  
2. Backend extracts and structures data  
3. Data is combined with live MongoDB metrics  
4. Sent to **Gemini 2.5 Flash**  
5. AI generates:
   - 🚨 Severity Analysis  
   - 📍 Resource Allocation Plan  
   - 👥 Volunteer Deployment Strategy  
   - 💡 Key Insights

## 🛠️ Technology Stack

| Layer | Technology |
| :--- | :--- |
| Frontend | React 19 + Vite |
| Styling | Tailwind CSS |
| Charts | Chart.js |
| Backend | Node.js + Express |
| Database | MongoDB + Mongoose |
| Authentication | Firebase |
| AI Engine | Google Gemini 2.5 Flash |
| File Processing | multer, pdf-parse, csv-parser |
